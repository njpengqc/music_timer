# \# 定时音乐播放器 - 项目说明

# 

# \## 📌 项目简介

# 

# 定时音乐播放器是一个基于Python和Tkinter的桌面应用程序，允许用户：

# \- 扫描指定文件夹中的音乐文件（支持MP3/FLAC/WAV格式）

# \- 设置定时播放任务

# \- 控制播放音量

# \- 管理音乐播放列表

# 

# \## 🛠️ 技术栈

# 

# \- \*\*Python 3.x\*\*

# \- \*\*Tkinter\*\* - GUI框架

# \- \*\*pygame\*\* - 音频播放

# \- \*\*mutagen\*\* - 音频元数据处理

# \- \*\*ttkthemes\*\* - 界面美化

# 

# \## 📦 安装与运行

# 

# \### 前提条件

# \- 安装Python 3.x

# \- 安装必要的依赖库：

# 

# ```bash

# pip install pygame mutagen ttkthemes

# ```

# 

# \### 运行程序

# 

# ```bash

# python main.py

# ```

# 

# \## 🎨 功能特点

# 

# 1\. \*\*音乐文件管理\*\*

# &nbsp;  - 浏览和选择音乐文件夹

# &nbsp;  - 自动扫描支持的音乐文件

# &nbsp;  - 右键菜单支持文件排序和删除

# 

# 2\. \*\*定时播放功能\*\*

# &nbsp;  - 设置精确的播放时间（时:分:秒）

# &nbsp;  - 设置播放时长（时:分:秒）

# &nbsp;  - 查看和管理所有定时任务

# 

# 3\. \*\*播放控制\*\*

# &nbsp;  - 立即播放功能

# &nbsp;  - 停止播放功能

# &nbsp;  - 实时音量控制

# 

# 4\. \*\*用户界面\*\*

# &nbsp;  - 现代化主题设计

# &nbsp;  - 响应式布局

# &nbsp;  - 状态栏显示播放信息

# &nbsp;  - 关于窗口包含版本信息

# 

# \## 📝 使用说明

# 

# 1\. \*\*添加音乐文件夹\*\*

# &nbsp;  - 点击"浏览"按钮选择包含音乐文件的文件夹

# &nbsp;  - 程序会自动扫描并显示所有支持的音乐文件

# 

# 2\. \*\*设置定时任务\*\*

# &nbsp;  - 在"播放时间"输入框中设置时间（格式：时:分:秒）

# &nbsp;  - 在"播放时长"输入框中设置时长（格式：时:分:秒）

# &nbsp;  - 点击"添加定时"按钮创建任务

# 

# 3\. \*\*立即播放\*\*

# &nbsp;  - 设置好播放时长后

# &nbsp;  - 点击"立即播放"按钮开始播放

# 

# 4\. \*\*管理任务\*\*

# &nbsp;  - 在"定时任务"标签页中查看所有任务

# &nbsp;  - 右键点击任务可以删除

# 

# \## ⚠️ 注意事项

# 

# 1\. 程序只支持MP3、FLAC和WAV格式的音频文件

# 2\. 时间格式必须严格遵循"时:分:秒"（例如：08:30:00）

# 3\. 播放时长最小为1秒

# 4\. 如果设置的时间已经过去，程序会自动设置为第二天的同一时间
